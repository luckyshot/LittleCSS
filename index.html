LittleCSS

Ultra-lean CSS library

LittleCSS is an ultra-lean CSS library of less than 9 KB (gzipped) that includes the necessary CSS classes and helpers for 99% of projects. It is heavily inspired by the best parts of TailwindCSS and Bootstrap frameworks.

All of the code is easily extendable and uses no pre-processors, taking advantage of latest modern CSS native features. This page you are seeing right now is using the default LittleCSS library.

  • ✅ Tiny: It is tiny so you don't need to worry about spliting files and so on
  • ✅ Plug'n'play: Add LittleCSS to your project and make it look nice from the start thanks to the default CSS
  • ✅ Hackable: The code itself is meant to be quick and easy to understand meaning you are encouraged to modify it instead of creating new selectors. If you are not going to use something, just remove it.
  • ✅ Native CSS: LESS and SCSS where lifesavers a few years ago, but nowadays vanilla CSS is very powerful and most browsers support the fun stuff
  • ✅ Simple: Most of the time we don't need compilers and build tools, by the time you need them your product will be at v3.0 and most other parts needing a refactoring too
